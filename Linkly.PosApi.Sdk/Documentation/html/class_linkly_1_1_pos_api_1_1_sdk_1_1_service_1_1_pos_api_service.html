Detailed Description

POS API service for communicating with the Cloud POS API v2. This service adopts an event-driven architecture to facilitate asynchronous communication with the API. Requests will be sent to the API in a thread and the result returned via the corresponding IPosApiEventListener method.

The user of this service must provided an implementation of IPosApiEventListener which handles all the methods accordingly.

To use this service the PIN pad must first be paired using PairingRequest or if the PIN pad was previously paired use SetPairSecret to set the pairing secret.
